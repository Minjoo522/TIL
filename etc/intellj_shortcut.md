# IntelliJ 단축키

| 단축키             | 📝                               |
| ------------------ | -------------------------------- |
| cmd + 1            | 프로젝트 목록 보기               |
| cmd + shift + o    | 파일 이름으로 검색               |
| cmd + opt + o      | 메소드 검색                      |
| cmd + l            | 특정 라인 번호로 이동            |
| opt + space        | 코드 구현부 즉시 보기            |
| cmd + alt + o      | 안 쓰는 임포트 삭제              |
| cmd + shift + z    | 작업 복구(redo)                  |
| cmd + shift + f    | 전체 검색                        |
| cmd + shift + r    | 전체 교체                        |
| cmd + r            | 현재 파일에서 교체               |
| shift + f6         | 이름 일괄 변경                   |
| cmd + shift + f6   | 타입 일괄 변경                   |
| ctrl + shift + j   | 라인 합치기                      |
| cmd + shift + ⬆️⬇️ | 구문 이동                        |
| shift + opt + ⬆️⬇️ | 라인 이동                        |
| opt + drag         | 멀티 포커스                      |
| opt + md + enter   | 위 엔터                          |
| shift + enter      | 아래 엔터                        |
| cmd + opt + l      | 코드 자동 정렬                   |
| cmd + shift + /    | 여러줄 주석                      |
| cmd + n            | getter, setter, 생성자 자동 생성 |
| ctrl + r           | run                              |
