# JavaScript 변수

## 변수(variable)

데이터를 저장할 때 쓰이는 '이름이 붙은 저장소'. 변수를 선언하고 값을 할당하면 메모리에 데이터가 저장된다.

### let vs const

- let

  - 모던한 변수 선언 키워드
  - 변수의 값을 변경할 수 있다(재할당 가능)

```javascript
let a; // 변수의 선언
a = 2; // 값 할당
a = 3; // 재할당
```

- const

  - 변수의 값을 변결할 수 없다(재할당 불가능)

```javascript
const b; // 변수의 선언
b = 2; // 값 할당
```

## 변수 이름 규칙

1. 문자, 숫자, $, \_만 사용 가능
2. 첫 글자는 숫자가 될 수 없다
3. 예약어는 변수 이름으로 사용할 수 없다

### 카멜 표기법(camelCase)

변수 이름을 만들 때는 일반적으로 카멜 표기법을 사용한다.

```javascript
// 바람직한 변수 이름 ⭕️
// 가급적 의미 있는 변수 이름을 사용하자❗️
let userName = 'Minjoo';
const myBirthday = '05.22.1994';

// 잘못된 변수 이름 ❌
let 1a; // 숫자로 시작할 수 없음
const user-name; // '-'을 사용할 수 없음
let function; // 예약어를 사용할 수 없음
```

### 대문자 상수

- 코드가 실행되기 전에 이미 값을 알고 있는 상수에 사용한다.
- '하드 코딩한' 값의 별칭을 만들 때 사용한다.
