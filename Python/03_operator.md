## 비교 연산자

| 연산자 | 📝          |
| :----- | :---------- |
| ==     | 같다        |
| !=     | 다르다      |
| <      | 작다        |
| \>     | 크다        |
| <=     | 작거나 같다 |
| \>=    | 크거나 같다 |

## 논리 연산자

| 연산자 | 📝                                |
| :----- | :-------------------------------- |
| not    | True➡️False, False➡️True          |
| and    | 피연산자가 모두 참일 때만 True    |
| or     | 피연산자 중 하나라도 참일 때 True |
