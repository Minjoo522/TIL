# 라이브러리

- gradle이나 maven 같은 빌드 툴들이 의존 관계 관리 해줌

## spring-boot-starter-web

- spring-boot-starter-tomcat : 웹서버
- spring-webmvc : 스프링 웹 MVC

## spring-boot-starter(공통)

- spring-boot

  - spring-core

- spring-boot-starter-logging
  - logback : 로그 출력 구현체
  - slf4j : 인터페이스

## 테스트 라이브러리

- junit : 테스트 프레임워크
- spring-test : 스프링 통합 테스트 지원
